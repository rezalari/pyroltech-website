# Pyroltech Website

Private preview of the new Pyroltech site.

- `index.html` , interactive WebGL hero prototype: particle "waste to clean" globe (Three.js + GSAP), neon atmosphere, region arcs, drag-to-spin, custom cursor, dark/light toggle, and the logo wall (placeholders).
- `SITEMAP.md` , full master sitemap (113+ pages, pillar-and-cluster, SEO/GEO/AEO).
- `templates/` , built page templates (e.g. `metallurgical-biocarbon.html`).

Story one-liner: **The world makes waste. We make it valuable.**

Self-contained, no build step. Open `index.html` in a modern browser, or serve via GitHub Pages.
